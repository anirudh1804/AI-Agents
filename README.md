A multi-agent system built with Phidata and Groq that combines web search and financial analysis capabilities to provide comprehensive market insights.
Features

Web Search Agent: Searches the web using DuckDuckGo for current information and news
Finance Agent: Analyzes stock data including prices, fundamentals, and analyst recommendations
Team Coordination: Intelligent agent orchestration for complex multi-step queries
Structured Output: Presents data in well-formatted tables and markdown
